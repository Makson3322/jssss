# Poltergeist Clone v4

## Install
```bash
npm init -y
npm i express socket.io
npm start
```

## Open
- Admin: `http://localhost:3000/admin.html?room=test`
- OBS: `http://localhost:3000/obs.html?room=test`

## Included
- Live / staging split
- Transparent OBS overlay
- Drag / resize / rotate
- Wheel zoom and middle-mouse pan
- QR object generation from text / URL
- Image upload from computer
- Image by URL
- Browser source URL normalization
- Timer inspector with start / pause / stop / reset / duration controls
- Room-scoped Socket.IO sync with in-memory state
